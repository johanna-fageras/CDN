## Installation

Choose one of the following options:

1. Link in HTML:

```html
<link
  rel="stylesheet"
  type="text/css"
  href="https://cdn.jsdelivr.net/gh/johanna-fageras/cdn/fonts/{fontname}.css"
/>
```

2. Import in CSS:

```css
@import url(https://cdn.jsdelivr.net/gh/johanna-fageras/cdn/fonts/{fontname}.css);
```
